# Federal Bureau of Investigation (FBI)This is a repo for managing machine readable artifacts for the [Federal Bureau of Investigation (FBI) APIs](https://www.fbi.gov/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Federal Bureau of Investigation (FBI) APIs](https://www.fbi.gov/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.fbi.gov/)- [Developer](https://www.fbi.gov/)- [Twitter](https://twitter.com/FBI)The [APIs.json](https://github.com/api-evangelist/federal-bureau-of-investigation-fbi/blob/master/apis.json) index provides links to all the relevant links for the Federal Bureau of Investigation (FBI) APIs, including the OpenAPI and Postman if they are present.
